PROJECTS

ascii-art-web

Ascii-art-web is a server-based program that allows users to create ASCII art banners using a web-based graphical user interface. The program implements HTTP endpoints for sending data to the Go server and displaying the results on the main page.

groupie-tracker

Groupie-tracker is a web-based program that receives data from a given API and displays information about bands and artists, including their name(s), image, the year they started, their first album, and members. The program creates a user-friendly website with data visualizations and allows for client-server communication for requesting information.

forum

Forum is a project that allows users to communicate with each other by creating posts and comments. The forum has categories associated with posts, and users can like or dislike posts and comments. There is also a filtering mechanism that allows users to filter posts by categories, created posts, and liked posts. The database library SQLite is used for data storage, and Docker is used for containerization.
